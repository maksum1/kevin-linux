1. Console Box (Cisco 2511) HowTo

CISCO 2511 acts as a console server which has 17 serial ports. All device in our lab can connect this 2511 via a serial port. For instance, you can access the FortiGate 400A device by using "telnet 172.22.14.201 2014" command where 172.22.14.201 is the 2511's IP address and 2014 is the specified port number.

There is only one user can access the tested device via the serial port at one time. In other words, when a user operates the device via the serial port, any others have to wait until the current user exits.

However, you can check the status of the serial port before accessing it via the way described below.


Step 1  

telnet 172.22.14.201

Step 2

Router> enable
Password: cisco
Router#

Enter "enable" mode. Enter the password "cisco".

You have entered enable mode when the prompt changes to "Router#".

Step 3   

Router# show line

This command will display all status of modem lines with a range from 1 to 17. The 17th line is the AUX line. The 1st line is corresponding to Line 2001 in the device. The 2nd line is corresponding to Line 2002 and so on. If a line is marked on (*) at its top, it indicates the line is used by someone currently. Also, if you exit the device incorrectly, the serial port will be also marked on (*) on its top.


Meanwhile, if you want to access the device, you can do as the following.

Step 1

telnet 172.22.4.225

Step 2

Router> enable
Password: cisco
Router#

Step 3

Router# clear <line number>

In addition, you can also verify the result using the "show line" command.
